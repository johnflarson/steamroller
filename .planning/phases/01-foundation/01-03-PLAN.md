const DIRECTIONS := [
    Vector2i(1, 0),   # horizontal
    Vector2i(0, 1),   # vertical
    Vector2i(1, 1),   # diagonal down-right
    Vector2i(1, -1),  # diagonal down-left

**_in_bounds(row: int, col: int) -> bool:** New function.
- Return `row >= 0 and row < rows and col >= 0 and col < cols`

**_collect_line(row: int, col: int, dir: Vector2i, player_idx: int) -> Array:** New function per RESEARCH.md Pattern 4.
- Start with cells array containing Vector2i(col, row) for the placed cell (the placed cell itself is NOT yet in scored_grid, so it's always included)
- Walk forward (row + dir.y, col + dir.x) while in bounds AND owner_grid[r][c] == player_idx AND scored_grid[r][c] == false — append each matching cell
- Walk backward (row - dir.y, col - dir.x) with same conditions — append each matching cell
- Return the cells array

CRITICAL per CONTEXT.md locked decision and RESEARCH.md Pitfall 6: The scored_grid check in _collect_line MUST skip cells where scored_grid[r][c] == true. This is the "spent cell" mechanic — cells that already contributed to a scored line cannot be reused.

**_check_score(row: int, col: int, player_idx: int) -> bool:** New function.
- For each direction in DIRECTIONS:
  - Call `_collect_line(row, col, dir, player_idx)`
  - If cells.size() >= 3:
    - Mark ALL cells in the line as scored: `scored_grid[c.y][c.x] = true` for each cell
    - Award 1 point: `players[player_idx].score += 1`
    - Log: `_log("%s scored! Line of %d detected. Score: %d" % [players[player_idx].name, cells.size(), players[player_idx].score])`
    - Return true immediately (per SCOR-02: max 1 point per turn — stop checking other directions)
- Return false (no line found)

**_check_win_or_stalemate() -> bool:** New function per RESEARCH.md code example.
1. Check win: `if players[current_player].score >= WIN_SCORE:`
   - `state = GameState.GAME_OVER`
   - `_log("Game over! %s wins with %d points!" % [players[current_player].name, players[current_player].score])`
   - `_disable_all_cells()`
   - `_update_ui()`
   - Return true
2. Check stalemate (board full): iterate all cells, if any owner_grid[r][c] == -1, set any_unclaimed = true
3. If not any_unclaimed:
   - `state = GameState.GAME_OVER`
   - Call `_resolve_stalemate()`
   - Return true
4. Return false

**Wire into _claim_cell():** Replace the placeholder comments from Plan 02.
- After setting owner_grid and coloring the button, BEFORE calling _advance_turn():
  1. `var scored := _check_score(row, col, current_player)`
  2. `if _check_win_or_stalemate(): return` (don't advance turn if game is over)
  3. Then call `_advance_turn()`

**Update _update_ui():** After scoring is live, ensure ScoresLabel shows current scores for all players clearly (e.g., "Player 1: 0 | Player 2: 1 | ...").
